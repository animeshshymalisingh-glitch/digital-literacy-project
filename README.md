## 📝 Module Summaries

### Task 1 – Digital Literacy Awareness Infographic (Module 1)
Created a single-page infographic using **Canva** covering three core topics: what digital literacy means, safe internet practices, and building a professional online presence. Designed for fellow first-year students to quickly grasp essential digital skills.

### Task 2 – Student Digital Portfolio (Module 2)
Set up accounts on **GitHub**, **LinkedIn**, and **Stack Overflow**. Each platform serves a specific purpose: GitHub for code repositories and projects, LinkedIn for professional networking, and Kaggle for data science practice. A profile README was created on GitHub.

### Task 3 – Coding & Collaboration Platforms (Module 3)
Completed the **"Solve Me First"** beginner challenge on HackerRank using Python 3. Have scored 115 points in HackerRank using Python3. Also created a 5-question **Digital Literacy Awareness Quiz** using Google Forms, linked to a Google Sheet for response collection.

🔗 **Digital Literacy Quiz (Google Form):** (https://docs.google.com/forms/d/e/1FAIpQLSdidqITEW9KykXcLOeRMzL04C-SiEjDHjlhZfaVgqxttamHPA/viewform?usp=publish-editor)

### Task 4 – Professional Email & Etiquette Guide (Module 4)
Drafted two professional emails — one requesting an assignment deadline extension from a professor, and one expressing interest in a summer internship. Created a Social Media Do's and Don'ts checklist with 7 points in each category.

### Task 5 – Cybercrime Awareness: UPI / Online Payment Fraud (Module 5)
Researched **UPI Payment Fraud** — one of the most common cybercrimes targeting Indian college students. Wrote a 220-word case study describing the step-by-step fraud mechanism and its consequences. Prepared a 10-point prevention checklist including 2 UPI-specific tips and official reporting channels.
